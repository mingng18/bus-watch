# BusWatch KL App Store Design

**Date:** 2026-07-12
**Status:** Approved

## Launch scope

- Product: BusWatch KL, a standalone Apple Watch transit app.
- Version/build: 1.0 (3).
- Price: Free.
- Availability: Malaysia only.
- Submission boundary: complete the listing and stop before the final Submit for Review action.

## Positioning

BusWatch KL provides quick Klang Valley bus information directly on Apple Watch. The listing will advertise only enabled and production-reachable release behavior: nearby bus stops, live bus positions, arrival estimates, and supported stop-by-stop bus progress.

The listing will not advertise favorites/home setup, service alerts, arrival reminders, an external journey planner, or other feature-flagged functionality disabled in build 3.

## Visual direction

The approved screenshot direction is **Clean Utility**:

- warm cream background;
- deep navy typography and borders;
- concise, high-contrast editorial headlines;
- real Watch UI as the main visual evidence;
- no fabricated routes, arrivals, locations, or unsupported interactions.

Screenshots will use Apple's accepted 416 × 496 Series 11/10 size consistently for the English localization.

## Metadata direction

- Name: BusWatch KL
- Subtitle: KL transit on your wrist
- Primary category: Navigation
- Secondary category: Travel
- Primary language: English (U.S.)
- Support and privacy URLs: public documents in the GitHub repository
- Review access: no sign-in, demo account, purchases, or subscriptions
- Release setting: manual release, preserving an explicit final approval gate

## Privacy and rating

The app requests precise location to perform a real-time nearby-transit lookup. Rider location is not retained by the backend, linked to an account, used for analytics/advertising, or shared for tracking. Under Apple's definition—data accessible longer than necessary to service a real-time request—the app declares **Data Not Collected**. The public privacy policy will still explain transient location transmission.

Age-rating answers are none/no for objectionable content, gambling, contests, unrestricted web access, and other mature-content categories. The app is not designated Made for Kids and does not override Apple's calculated rating.

## Review notes

Review notes will state that BusWatch KL is a standalone Watch app whose iOS target is packaging-only. They will explain location permission, provide a Kuala Lumpur simulator coordinate for testing, describe the live map and rail-arrival paths, and note that no login is required.
